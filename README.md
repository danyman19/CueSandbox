# CueSandbox

Working on a sandbox mode for cue cards

Current features:
- Card drawing
- Proper card rotation
- All 18 cards from lol/oceans working
- Stored round/turn scores

In development:

- A way to create own deck from a pool, and set orderings
- Way to set how "random" cards work without hard coding
- Add more cards/decks
